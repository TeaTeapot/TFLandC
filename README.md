Задание «Разработка пользовательского интерфейса (GUI) для языкового процессора»


Цель работы: Разработать приложение – текстовый редактор с графическим интерфейсом пользователя, который в дальнейшем будет дополнен функциями языкового процессора для анализа исходного кода.


Выполнил студент группы АВТ-314 Гладышев Р.М.


Описание проекта: Реализовано оконное приложение – текстовый редактор, являющийся основой для будущего языкового процессора.
Реализовано:
Главное меню программы
Панель инструментов
Область редактирования текста
Область отображения результатов (для будущего компилятора)
Реализован функционал меню "Файл", "Правка" и "Справка". Реализация меню "Текст" и "Пуск" отложена до надобности.


Используемые технологии:
Язык программирования: C#
GUI: Windows Forms
Версия .NET: .NET Framework 4.7.2 / .NET 8.0
